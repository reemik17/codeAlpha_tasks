# CodeAlpha C++ Projects
This repository contains the C++ projects I completed during the CodeAlpha C++ Internship. These projects are designed to strengthen core programming skills through practical 
applications and demonstrate concepts such as object-oriented programming, file handling, functions, classes, and problem-solving.

## Projects
### 1. CGPA Calculator
A console-based application that calculates semester GPA and overall CGPA based on course grades and credit hours.

**Features:**
* Input multiple courses
* Calculate GPA and CGPA
* Display course-wise grades and credits
* Compute total credits and grade points

### 2. Login & Registration System
A user authentication system that allows users to register and log in using file-based storage.

**Features:**
* User registration
* Login authentication
* Duplicate username validation
* Store user credentials using files

### 3. Banking System
A simple banking application that manages customers, accounts, and transactions.

**Features:**
* Create customer accounts
* Deposit and withdraw money
* Transfer funds between accounts
* View account balance
* Display transaction history

## Technologies Used
* C++
* Object-Oriented Programming (OOP)
* File Handling

## Learning Outcomes
Through these projects, I improved my understanding of:
* C++ programming fundamentals
* Object-oriented design
* File input/output operations
* Data management
* Debugging and problem-solving

These projects helped me gain practical experience in developing structured, interactive, and user-oriented applications using C++.
