# Description
This project is a simple console-based CGPA Calculator developed in C++. It allows users to enter course grades and credit hours,
calculates the semester GPA, and computes the overall CGPA using previous academic records. The program validates user input and displays
detailed results for each course.

# Features
* Enter multiple courses
* Input grade points and credit hours
* Validate grade points (0–4 scale)
* Validate credit hours
* Calculate total credits and grade points
* Compute semester GPA
* Calculate overall CGPA using previous records
* Display individual course details and final results

# Concepts Practiced
* Arrays
* Loops
* Conditional Statements
* User Input/Output
* Arithmetic Calculations
* Input Validation
