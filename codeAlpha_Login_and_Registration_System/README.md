# Description
This project is a simple console-based Login and Registration System developed in C++. It allows users to register with a username
and password, stores credentials in a file, and provides a login feature to verify user identity. The program includes input validation,
duplicate username checking, and file handling for persistent data storage.

# Features
* User registration with username and password
* Login authentication system
* Validation for empty inputs
* Duplicate username detection
* error messages
  
# Concepts Practiced
* Functions
* File Handling (ifstream, ofstream)
* User Input/Output
* Conditional Statements
* Loops
* String
