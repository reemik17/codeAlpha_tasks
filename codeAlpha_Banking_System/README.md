# Banking System
## Description
This project is a simple console-based banking system developed in C++. It allows users to create bank accounts, deposit and withdraw money, transfer funds between accounts, view account details, and check transaction history. The program demonstrates the use of object-oriented programming concepts to simulate basic banking operations.

## Features

* Create new customer accounts
* Deposit money into an account
* Withdraw money with balance validation
* Transfer funds between accounts
* View account information and balance
* Display transaction history
* Prevent duplicate account numbers
* Handle invalid inputs gracefully

## Concepts Practiced

* Object-Oriented Programming (OOP)
* Classes and Objects
* Functions
* Arrays
* Loops
* Conditional Statements
* String Manipulation
* Menu-Driven Programming
* Input Validation
