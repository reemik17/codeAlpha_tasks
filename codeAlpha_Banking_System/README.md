# Banking System

## Description
The **Banking System** is a console-based application developed in **C++** as part of the **CodeAlpha C++ Internship**. The project simulates basic banking operations, allowing users to create accounts, perform transactions, and manage account information through a menu-driven interface. It demonstrates the implementation of object-oriented programming concepts and fundamental C++ programming techniques.

## Features
* Create new customer accounts
* Deposit money into an account
* Withdraw money with balance validation
* Transfer funds between accounts
* View account details and current balance
* Display transaction history
* Prevent duplicate account numbers
* Handle invalid account numbers and invalid transaction amounts
* Menu-driven console interface

## Project Structure
The application consists of three main classes:

* **Customer** – Stores customer information such as name and account number.
* **Transaction** – Records transaction type, amount, and updated balance.
* **Account** – Manages account creation, deposits, withdrawals, fund transfers, balance inquiries, and transaction history.
